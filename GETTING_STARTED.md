(~64KB chunks)\n- Zero-copy texture transfer (CVMetalTextureCache)\n- Real-time statistics overlay\n\n### Deployment\n- **Target**: visionOS 26\n- **Minimum**: visionOS 2.0+\n- **Swift**: 5.9+\n- RawStreamEngine]\n    ↓ Annex-B NAL units + PTS\n[VideoDecoder] ← VideoToolbox hardware\n    ↓ CVPixelBuffer (NV12)\n[MetalVideoRenderer] ← GPU rendering\n    ↓ RGB texture\n[SwiftUI PlayerScreen]\n    ↓\n  User sees video\n```\n\n## Testing Checklist\n\nOnce built:\n- [ ] App launches (RootView shows)\n- [ ] Video plays (tap media item)\n- [ ] Controls appear (tap fullscreen player)\n- [ ] Stats
