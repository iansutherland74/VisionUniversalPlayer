H.264 NAL unit types (SPS, PPS, IDR slice, etc.)\n- `HEVCNALUnitType` - HEVC NAL unit types (VPS, SPS, PPS, etc.)\n\nStructs:\n- `NALUnit` - Parsed H.264 unit\n- `HEVCNALUnit` - Parsed HEVC unit\n\nUse: NALParser returns these to identify NAL boundaries.\n\n### NALParser.swift\n**Annex-B stream parsing**\n\nClass: `NALParser`\n\nMethods:\n- `parseAnnexBStream(Data) -> [NALUnit]`\n- `parseHEVCAnnexBStream(Data) -> [HEVCNALUnit]`\n\nResponsibilities:\n- Find Annex-B start codes (00 00 01 / 00 00 00 01)\n- Extract complete NAL units\n- Identify unit types\n\nUse: Call when receiving raw Annex-B data.\n\n### VideoDecoder.swift\n**VideoToolbox hardware decoder wrapper**\n\nClass: `VideoDecoder`\n\nMethods:\n- `decodeAnnexBH264(Data, CMTime)`\n- `decodeAnnexBHEVC(Data, CMTime)`\n- `flush()` - finish pending frames\n- `reset()` - clear session\n\nDelegate Protocol: `VideoDecoderDelegate`\n- `decoderDidProducePixelBuffer(_:pts:)`\n- `decoderDidUpdatePixelDimensions(width:height:)`\n- `decoderDidEncounterError(_:)`\n\nResponsibilities:\n- Create VTDecompressionSession from SPS/PPS/VPS\n- Decode NAL slices → CVPixelBuffer\n- Handle format changes\n\n### FFmpegBridge.h\n**C header for FFmpeg functions**\n\n```c\nint ffmpeg_open(const char *url);\nint ffmpeg_read_annexb_packet(int handle, uint8_t **data, int *size, double *ptsSeconds);\nvoid ffmpeg_free_packet(uint8_t *data);\nvoid ffmpeg_close(int handle);\n```\n\nResponsibilities:\n- Expose FFmpeg C API to Swift\n- Handle remote URLs (HTTP/FTP/WebDAV)\n- Auto-apply bitstream filters\n\n### FFmpegBridge.c\n**C implementation of FFmpeg bridge**\n\nKey Functions:\n- `ffmpeg_open()` - Open container, find video stream, init bitstream filter\n- `ffmpeg_read_annexb_packet()` - Read next Annex-B frame + PTS\n- `ffmpeg_close()` - Clean up\n\nInternal:\n- Thread-safe handle array (pthread mutex)\n- Max 16 concurrent streams\n- Auto-detects H.264 vs HEVC\n\n### FFmpegDemuxer.swift\n**Swift wrapper around FFmpeg bridge**\n\nClass: `FFmpegDemuxer`\n\nInitializer:\n```swift\ninit(url: URL, codec: CodecType) throws\n```\n\nMethods:\n- `nextPacket() async throws -> (Data, CMTime)?`\n- `close()`\n\nResponsibilities:\n- Wrap C function calls\n- Convert C errors to Swift exceptions\n- Stream Annex-B packets with PTS\n\n### FFmpegEngine.swift\n**Combines FFmpegDemuxer + VideoDecoder**\n\nConforms to: `VideoOutputEngine`\n\nPublishers:\n- `pixelBufferPublisher` - Decoded video frames\n- `dimensionPublisher` - Resolution changes\n\nMethods:\n- `start(item:) async` - Begin streaming\n- `stop()` - Stop and cleanup\n\nResponsibilities:\n- Run demuxer in background task\n- Send NAL units to decoder\n- Forward pixels to subscribers\n\n### RawStreamEngine.swift\n**Raw Annex-B stream decoder**\n\nConforms to: `VideoOutputEngine`\n\nResponsibilities:\n- Download raw bytes via URLSession\n- Parse Annex-B without container\n- Handle incomplete NAL units in StreamBuffer\n- Synthesize PTS from frame counter\n\nUse: For raw H.264/HEVC Annex-B streams (no container).\n\n### HLSClient.swift\n**HLS playlist parsing**\n\nClass: `HLSClient`\n\n```swift\nfunc fetchPlaylist() async throws -> HLSPlaylist\nfunc fetchSegment(url: URL) async throws -> Data\n```\n\nTypes:\n- `HLSPlaylist` - Parsed m3u8 metadata\n- `HLSSegment` - TS/fMP4 segment info\n\nNote: Used for m3u8 parsing; FFmpeg handles actual playback.\n\n### PlayerViewModel.swift\n**Engine orchestration and statistics**\n\n```swift\n@Observable\nfinal class PlayerViewModel\n```\n\nMethods:\n- `playMedia(_:) async` - Choose engine, start playback\n- `stopPlayback() async` - Clean shutdown\n- `togglePlayPause()`\n\nPublishers:\n- `pixelBufferPublisher` - Main video output\n\nResponsibilities:\n- Create appropriate engine (raw vs FFmpeg)\n- Subscribe to engine outputs\n- Forward to UI subscribers\n\n### MetalVideoRenderer.swift\n**GPU rendering pipeline**\n\nClass: `MetalVideoRenderer`\n\nMethods:\n- `render(pixelBuffer:to:in:)` - Render frame to drawable\n- `flushTextureCache()`\n\nResponsibilities:\n- Create CVMetalTextureCache\n- Build render pipeline state\n- Compile NV12→RGB shader\n- Issue render commands\n\n### MetalHostView.swift\n**MTKView wrapper for VisionOS**\n\nClass: `MetalHostView: NSObject, ObservableObject`\n\nMethods:\n- `updatePixelBuffer(_:)` - Update video frame\n- `getMetalView() -> MTKView?` - Access underlying view\n\nResponsibilities:\n- Manage MTKView lifecycle\n- Implement MTKViewDelegate\n- Handle drawing\n\n### Shaders.metal\n**Metal shading language**\n\nVertex Shader: `vertexShader()`\n- Passthrough quad (-1,-1) to (1,1)\n- Pass texCoords\n\nFragment Shader: `fragmentShader()`\n- Sample Y plane (texture 0) and UV planes (texture 1)\n- Convert NV12 YUV→RGB (BT.709)\n- Output RGBA\n\n### MetalVideoView.swift\n**SwiftUI view for Metal rendering**\n\n```swift\nstruct MetalVideoView: View\n```\n\nBindings:\n- Input: `pixelBufferPublisher`\n- Manages: `MetalHostView` + `MetalViewContainer`\n\nResponsibilities:\n- Display Metal view in SwiftUI hierarchy\n- Subscribe to pixel buffer updates\n- Forward to Metal renderer\n\n### RootView.swift\n**Home screen with media browser**\n\n```swift\nstruct RootView: View\n```\n\nUI Elements:\n- Hero banner (welcome text)\n- \"Featured\" media row\n- \"Recent\" media row\n- Navigation to DetailView\n\nResponsibilities:\n- Display all available media\n- Navigate to detail pages\n- Handle media selection\n\n### DetailView.swift\n**Media detail and preview page**\n\n```swift\nstruct DetailView: View\n```\n\nUI Elements:\n- Thumbnail image\n- Title, description\n- Codec and duration badges\n- Play button\n\nBinding: `onPlay` callback → starts PlayerScreen\n\n### PlayerScreen.swift\n**Fullscreen video player**\n\n```swift\nstruct PlayerScreen: View\n```\n\nUI Layers:\n1. HUD overlay (stats)\n\nGestures:\n- **Tap**: Toggle controls visibility\n- **Long press**: Toggle HUD\n\nLifecycle:\n- `onTask`: Start playback\n- `onDisappear`: Stop playback\n\n### [FFmpegDemuxer]\n  ↓ FFmpeg reads container\n  ↓ Apply h264_mp4toannexb filter\n  ↓ [VideoDecoder]\n  ↓ VideoToolbox decodes NALs\n  ↓ CVPixelBuffer (NV12)\n  ↓ [MetalVideoRenderer]\n  ↓ Metal fragment shader YUV→RGB\n  ↓ [MetalVideoView]\n  ↓ SwiftUI displays on screen\n```\n\n### Raw Stream Playback\n```\nMediaItem (raw Annex-B URL)\n  ↓\nRawStreamEngine.start()\n  ↓ [URLSession]\n  ↓ Download bytes\n  ↓ [NALParser]\n  ↓ Find Annex-B NAL boundaries\n  ↓ [VideoDecoder]\n  ↓ VideoToolbox decodes\n  ↓ [MetalVideoRenderer]\n  ↓ GPU rendering\n  ↓ Display\n```\n\n---\n\n## Key Interfaces (Protocols)\n\n### VideoOutputEngine\n```swift\nprotocol VideoOutputEngine: AnyObject {\n    var pixelBufferPublisher: AnyPublisher<CVPixelBuffer, Never> { get }\n    var dimensionPublisher: AnyPublisher<(width: Int32, height: Int32), Never> { get }\n    func start(item: MediaItem) async\n    func stop()\n}\n\n// Implemented by: FFmpegEngine, RawStreamEngine\n```\n\n### VideoDecoderDelegate\n```swift\nprotocol VideoDecoderDelegate: AnyObject {\n    func decoderDidProducePixelBuffer(_ pixelBuffer: CVPixelBuffer, pts: CMTime)\n    func decoderDidUpdatePixelDimensions(width: Int32, height: Int32)\n    func decoderDidEncounterError(_ error: Error)\n}\n\n// Implemented by: FFmpegEngine, RawStreamEngine\n```\n\n###
